Secretaria Escolar

Repositório para criação de avaliação de notas para Secretaria Escolar.
O projeto tem como intuito criar um sistema de Secretaria Escolar simplificado para avaliação de Aprovação/Reprovação de alunos, a partir do recebimento e cálculo das notas avaliativas individuais, além de gerar um indicador de média coletiva por turma, para melhor análize de desenpenho.

O código fonte do projeto encontra-se neste repositório
https://github.com/lorenagbatista/Secretaria-Escolar/blob/main/secretaria_escolar

O projeto pode ser executado por meio do compilador de pseudocódigo Portugol Webstudio, acesso: https://portugol.dev/
